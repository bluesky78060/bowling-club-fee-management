1 앱 소개 및 주요 기능

볼링 동호회 회비 관리 앱은 볼링 동호회 운영에 필요한 모든 관리 기능을 한 곳에서 편리하게 사용할 수 있도록 설계되었습니다. 회원 관리부터 회비 정산, 게임 점수 기록까지 동호회 총무님의 업무를 효율적으로 도와드립니다.

🎯 이런 분들께 추천합니다

  • 볼링 동호회 총무 또는 회계 담당자
  • 매 모임마다 회비 정산이 번거로우신 분
  • 회원별 에버리지, 핸디캡을 체계적으로 관리하고 싶은 분
  • 엑셀 대신 간편한 모바일 앱으로 관리하고 싶은 분

✨ 주요 기능

👥 회원 관리

  • 회원 정보 등록/수정/삭제
  • 활동/휴면/탈퇴 상태 관리
  • 이름, 연락처 검색
  • 성별별 아바타 자동 생성

💰 회비 정산

  • 모임별 회비 자동 계산
  • 게임 수에 따른 게임비 계산
  • 감면 대상자 할인 적용
  • 정산 내역 히스토리 관리

🎳 점수 기록

  • 게임별 점수 기록
  • 에버리지 자동 계산
  • 핸디캡 관리
  • 성적 추이 확인

⚙️ 편의 기능

  • 데이터베이스 백업/복원
  • 설정 내보내기/가져오기
  • 연락처 자동 포맷팅
  • 직관적인 UI/UX

💡 시작하기 전에

앱을 처음 사용하시나요? 다음 순서로 설정하시면 편리합니다:

  • 설정에서 동호회 이름, 회비, 게임비를 먼저 설정하세요.
  • 회원 관리에서 동호회 회원들을 등록하세요.
  • 이제 정산 관리에서 모임별 정산을 시작할 수 있습니다!

3 회원 관리

회원 관리 화면에서는 동호회 회원들의 정보를 등록하고 관리할 수 있습니다. 직관적인 제스처를 통해 빠르고 편리하게 회원 정보를 관리하세요.

📋 3.1 회원 목록

검색 기능

화면 상단의 검색창에서 회원을 빠르게 찾을 수 있습니다. 이름의 일부 또는 전화번호 일부만 입력해도 검색됩니다.

  • 예시: "김" 입력 → 김씨 성을 가진 모든 회원 표시
  • 예시: "9339" 입력 → 해당 번호가 포함된 회원 표시

필터 기능

검색창 아래의 필터 칩을 눌러 회원을 분류해서 볼 수 있습니다.

  • 활동: 현재 활동 중인 회원만 표시
  • 휴면: 휴면 상태인 회원만 표시
  • 전체: 모든 회원 표시

회원 추가

필터 칩 옆의 "+ 추가" 버튼을 눌러 새 회원을 등록할 수 있습니다.

👆 제스처 사용법

회원 카드에서 다양한 제스처를 사용하여 빠르게 작업할 수 있습니다.

👆

탭 (터치)

회원 카드를 한 번 터치하면 회원 상세 정보 화면으로 이동합니다. 회원의 모든 정보와 게임 기록, 정산 내역을 확인할 수 있습니다.

👆⏱️

길게 누르기 (롱프레스)

회원 카드를 1초 이상 길게 누르면 옵션 메뉴가 나타납니다.

  • 상세 보기: 회원 상세 정보 확인
  • 수정: 회원 정보 수정 화면 이동
  • 삭제: 회원 삭제 (확인 필요)
👈

왼쪽으로 스와이프

회원 카드를 왼쪽으로 밀면 빨간색 삭제 버튼이 나타납니다. 계속 밀거나 삭제 버튼을 터치하면 삭제 확인 창이 표시됩니다.

👉

오른쪽으로 스와이프

회원 카드를 오른쪽으로 밀면 파란색 수정 버튼이 나타납니다. 계속 밀거나 수정 버튼을 터치하면 회원 수정 화면으로 이동합니다.

📝 3.2 회원 등록

새 회원을 등록할 때 입력하는 항목들입니다. 필수 항목만 입력해도 등록이 가능합니다.

항목 구분 설명 예시
이름 필수 회원의 이름 (2~10자) 홍길동
연락처 필수 휴대폰 번호 (자동 하이픈 입력) 010-1234-5678
성별 선택 남성/여성 선택 (아바타 색상 결정) 남성 → 파란색, 여성 → 분홍색
가입일 선택 동호회 가입 날짜 2024년 1월 15일
초기 에버리지 선택 가입 시점의 에버리지 점수 150
핸디캡 선택 핸디캡 점수 (보통 0) 0
상태 선택 활동/휴면/탈퇴 상태 활동
감면 대상자 선택 체크 시 게임비 50% 할인 체크/해제
메모 선택 기타 참고 사항 목요일 불참

💡 연락처 입력 팁

연락처 입력 시 숫자만 입력하면 됩니다. 01012345678 → 자동으로 010-1234-5678 형식으로 표시됩니다. 하이픈(-)을 직접 입력할 필요가 없습니다!

✏️ 3.3 회원 수정/삭제

회원 정보 수정

회원 카드를 오른쪽으로 스와이프

또는 길게 눌러서 "수정" 메뉴를 선택합니다.

수정할 정보 변경

회원 등록 화면과 동일한 형식으로 정보를 수정합니다.

"저장" 버튼 클릭

변경사항이 저장되고 회원 목록으로 돌아갑니다.

회원 삭제

회원 카드를 왼쪽으로 스와이프

또는 길게 눌러서 "삭제" 메뉴를 선택합니다.

삭제 확인

"삭제하시겠습니까?" 확인 창에서 "삭제" 버튼을 누릅니다.

⚠️ 주의: 회원 삭제 시

회원을 삭제하면 해당 회원의 모든 데이터(정산 기록, 게임 점수 등)가 함께 삭제됩니다. 삭제된 데이터는 복구할 수 없습니다. 장기간 불참하는 회원은 삭제 대신 "휴면" 상태로 변경하는 것을 권장합니다.

🏷️ 3.4 회원 상태 관리

회원은 3가지 상태로 관리됩니다. 상태에 따라 정산 시 자동 처리가 달라집니다.

✅ 활동

정상적으로 활동 중인 회원

  • 정산 시 참석자 목록에 표시
  • 회원 목록 기본 필터에 포함
  • 통계에 반영

⏸️ 휴면

일시적으로 활동을 쉬는 회원

  • 정산 시 참석자 목록에서 제외
  • "휴면" 필터에서 확인 가능
  • 데이터는 그대로 보존

❌ 탈퇴

동호회를 탈퇴한 회원

  • 정산 시 표시되지 않음
  • 기록 보존 (히스토리용)
  • 완전 삭제는 별도 처리

💡 상태 변경은 언제 하나요?

  • 휴면으로 변경: 개인 사정으로 3개월 이상 불참이 예상될 때
  • 활동으로 복귀: 휴면 회원이 다시 모임에 참석할 때
  • 탈퇴 처리: 회원이 공식적으로 탈퇴 의사를 밝혔을 때

4 정산 관리

정산 관리 화면에서는 모임별로 회비와 게임비를 자동 계산하고, 참석자별 납부 금액을 관리할 수 있습니다.

📋 4.1 새 정산 만들기

정산 화면에서 "새 정산" 버튼 클릭

정산 목록 화면 하단의 파란색 "새 정산" 버튼을 누릅니다.

모임 날짜 선택

정산할 모임의 날짜를 캘린더에서 선택합니다. 기본값은 오늘 날짜입니다.

참석 회원 선택

모임에 참석한 회원들을 체크박스로 선택합니다. 활동 상태인 회원만 목록에 표시됩니다.

회원별 게임 수 입력

각 참석 회원이 친 게임 수를 입력합니다. +/- 버튼 또는 직접 입력이 가능합니다.

자동 계산 확인

회비 + 게임비가 자동으로 계산되어 실시간으로 표시됩니다.

"정산 완료" 버튼 클릭

모든 입력이 완료되면 정산을 저장합니다.

🧮 4.2 정산 계산 방식

정산 금액은 다음과 같이 자동 계산됩니다.

💰 정산 계산 예시

회원 "홍길동"님이 4게임을 친 경우:

기본 회비 10,000원
게임비 (4게임 × 3,500원) 14,000원
총 납부 금액 24,000원

💵 기본 회비

설정에서 지정한 기본 회비 금액이 모든 참석자에게 동일하게 적용됩니다.

  • 기본값: 10,000원
  • 설정에서 변경 가능
  • 모든 참석자 동일 적용

🎳 게임비

게임 수 × 1게임당 게임비로 계산됩니다. 회원마다 게임 수가 다를 수 있습니다.

  • 기본값: 3,500원/게임
  • 설정에서 변경 가능
  • 개인별 게임 수 입력

🏷️ 4.3 감면 적용

감면 대상자로 등록된 회원은 게임비가 자동으로 50% 할인됩니다. 기본 회비는 할인되지 않습니다.

💰 감면 대상자 계산 예시

감면 대상자 "김영희"님이 4게임을 친 경우:

기본 회비 10,000원
게임비 (4게임 × 3,500원 × 50%) 7,000원
총 납부 금액 17,000원

💡 감면 대상자 설정 방법

회원 등록/수정 화면에서 "감면 대상자" 체크박스를 선택하면 됩니다. 65세 이상 어르신, 장애인, 기초생활수급자 등 동호회 규정에 따라 적용해 주세요.

5 모임 및 점수 기록

각 모임에서 회원들의 게임 점수를 기록하고, 에버리지와 핸디캡을 자동으로 관리할 수 있습니다.

🎯 5.1 점수 입력

정산 상세 화면에서 "점수 기록" 탭 선택

정산 완료 후 해당 정산을 열어 점수 기록 탭으로 이동합니다.

회원 선택

점수를 입력할 회원을 선택합니다.

게임별 점수 입력

1게임, 2게임, 3게임... 순서대로 점수를 입력합니다.

자동 에버리지 계산 확인

입력된 점수로 에버리지가 자동 계산되어 표시됩니다.

📊 5.2 에버리지 계산

에버리지는 설정에서 지정한 게임 수의 평균으로 계산됩니다.

📊 에버리지 계산 예시

에버리지 계산 게임 수가 12게임으로 설정된 경우:

최근 12게임 점수 합계 1,920점
게임 수 12게임
에버리지 160점

📈 에버리지

최근 N게임의 평균 점수입니다. 실력 수준을 나타내는 지표로 사용됩니다.

  • 기본 계산 게임 수: 12게임
  • 설정에서 변경 가능
  • 점수 입력 시 자동 갱신

🎯 핸디캡

실력 차이를 보정하기 위한 점수입니다. 에버리지가 낮은 회원에게 부여됩니다.

  • 기본값: 0점
  • 상한선 설정 가능
  • 회원 정보에서 직접 설정

💡 핸디캡 계산 공식 예시

일반적인 핸디캡 공식: (200 - 에버리지) × 0.8
예) 에버리지 160점인 회원: (200 - 160) × 0.8 = 32점 핸디캡

7 자주 묻는 질문 (FAQ)

Q. 회원을 실수로 삭제했어요. 복구할 수 있나요?

삭제된 회원은 직접 복구할 수 없습니다. 다만, 이전에 백업해 둔 데이터베이스 파일이 있다면 복원을 통해 되돌릴 수 있습니다. 정기적인 백업을 권장드립니다.

Q. 휴면 회원도 정산에 포함시킬 수 있나요?

기본적으로 휴면 회원은 정산 시 참석자 목록에 표시되지 않습니다. 특별히 포함시키려면 해당 회원의 상태를 "활동"으로 변경한 후 정산을 진행하세요.

Q. 게임비를 회원마다 다르게 설정할 수 있나요?

현재 버전에서는 모든 회원에게 동일한 게임비가 적용됩니다. 다만, "감면 대상자"로 설정하면 게임비가 50% 할인됩니다.

Q. 백업 파일은 어디에 저장하는 게 좋나요?

Google 드라이브나 원드라이브 같은 클라우드 저장소에 백업하시면 휴대폰을 분실하거나 교체해도 데이터를 복원할 수 있습니다.

Q. 여러 기기에서 동시에 사용할 수 있나요?

현재 버전은 단일 기기에서 사용하도록 설계되었습니다. 데이터를 다른 기기로 옮기려면 백업 → 복원 과정을 거쳐야 합니다.

Q. 연락처에 하이픈(-)을 넣어야 하나요?

아니요, 숫자만 입력하시면 됩니다. 예를 들어 "01012345678"을 입력하면 자동으로 "010-1234-5678" 형식으로 표시됩니다.

Q. 이전 정산 내역을 수정할 수 있나요?

네, 정산 목록에서 수정하려는 정산을 선택한 후 편집 모드로 들어가면 참석자, 게임 수 등을 수정할 수 있습니다.

8 문제 해결

🔧 일반적인 문제

문제: 앱이 갑자기 종료됩니다

해결 방법:
1. 앱을 완전히 종료 후 다시 실행해 보세요.
2. 휴대폰을 재부팅해 보세요.
3. 앱 캐시를 삭제해 보세요. (설정 > 앱 > 볼링 회비 관리 > 저장공간 > 캐시 삭제)
4. 문제가 지속되면 백업 후 앱을 재설치하세요.

문제: 회원 검색이 안 됩니다

해결 방법:
1. 검색어에 오타가 없는지 확인하세요.
2. 현재 필터가 "전체"로 설정되어 있는지 확인하세요.
3. 검색창을 비우고 다시 검색해 보세요.

문제: 백업 파일을 찾을 수 없습니다

해결 방법:
1. 파일 관리자 앱에서 "다운로드" 폴더를 확인하세요.
2. "bowling_db_" 또는 "bowling_settings_"로 시작하는 파일을 찾으세요.
3. Google 드라이브에 저장했다면 드라이브 앱에서 확인하세요.

문제: 복원 후 데이터가 안 보입니다

해결 방법:
1. 앱을 완전히 종료 후 다시 실행하세요.
2. 복원한 백업 파일이 올바른 파일인지 확인하세요.
3. 백업 파일이 손상되지 않았는지 확인하세요.

💬 추가 지원이 필요하신가요?

위의 방법으로도 해결되지 않는 문제가 있다면, 문제 상황을 상세히 기록하여 개발자에게 문의해 주세요. 백업 파일과 함께 문의하시면 더 빠른 해결이 가능합니다.
