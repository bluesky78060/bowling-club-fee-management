개인정보처리방침

볼링 동호회 회비 관리

시행일: 2025년 1월 17일

핵심 요약: 본 앱은 모든 데이터를 사용자의 기기에만 저장하며, 외부 서버로 개인정보를 전송하지 않습니다.

제1조 (목적)

「볼링 동호회 회비 관리」(이하 "앱")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다. 본 개인정보처리방침은 앱이 수집하는 정보와 그 이용 방법에 대해 안내합니다.

제2조 (수집하는 개인정보 항목)

앱은 다음과 같은 정보를 수집합니다:

1. 사용자가 직접 입력하는 정보

  • 회원 정보: 이름, 기본 에버리지
  • 모임 정보: 모임 일자, 장소, 참석자
  • 점수 정보: 게임별 점수
  • 정산 정보: 회비, 식비, 벌금 등
  • 찬조 정보: 후원자명, 금액

2. 카메라 사용 시 수집되는 정보

  • 점수표 이미지 (OCR 인식용, 임시 저장 후 삭제)
  • 영수증 이미지 (OCR 인식용, 임시 저장 후 삭제)
참고: 촬영된 이미지는 OCR 처리 후 즉시 삭제되며, 기기 외부로 전송되지 않습니다.

제3조 (개인정보의 처리 목적)

수집된 정보는 다음 목적으로만 사용됩니다:

  • 볼링 동호회 회원 관리
  • 모임 및 점수 기록 관리
  • 회비 정산 및 재정 관리
  • 통계 및 랭킹 제공

제4조 (개인정보의 보관 및 파기)

항목 보관 위치 보관 기간
모든 앱 데이터 사용자 기기 내부 저장소 앱 삭제 시까지
OCR 이미지 기기 임시 메모리 처리 완료 후 즉시 삭제

사용자가 앱을 삭제하면 모든 데이터가 기기에서 완전히 삭제됩니다.

제5조 (개인정보의 제3자 제공)

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 모든 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

제6조 (개인정보의 안전성 확보조치)

앱은 개인정보 보호를 위해 다음과 같은 조치를 취하고 있습니다:

  • 로컬 저장: 모든 데이터는 사용자 기기에만 저장
  • 네트워크 미사용: 인터넷 연결 없이 작동 (OCR 기능 제외)
  • 암호화: Android 시스템의 기본 암호화 적용

제7조 (이용자의 권리)

이용자는 다음과 같은 권리를 행사할 수 있습니다:

  • 열람권: 앱 내에서 모든 저장된 데이터 확인 가능
  • 정정권: 앱 내에서 언제든지 데이터 수정 가능
  • 삭제권: 앱 내에서 개별 데이터 삭제 또는 앱 삭제로 전체 데이터 삭제 가능
  • 이동권: 백업 기능을 통해 데이터 내보내기 가능

제8조 (앱 접근 권한)

권한 필수 여부 사용 목적
카메라 선택 점수표/영수증 OCR 스캔
저장소 선택 데이터 백업/복원

선택 권한은 해당 기능 사용 시에만 요청되며, 허용하지 않아도 기본 기능 사용이 가능합니다.

제9조 (개인정보처리방침 변경)

본 개인정보처리방침이 변경되는 경우, 앱 업데이트 또는 앱 내 공지를 통해 안내드립니다.
